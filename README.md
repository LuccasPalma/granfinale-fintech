# 💳 Fintech FIAP - Via Cartão

Sistema web de gerenciamento financeiro pessoal desenvolvido para a FIAP utilizando **Spring Boot** no backend e **ReactJS** no frontend.

# ✨ Funcionalidades

✅ Autenticação de usuário  
✅ Dashboard financeiro  
✅ CRUD de contas

# ⚙️ Backend (Spring Boot)

## Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- Oracle Database

---

## ▶️ Como executar o backend

Abra a pasta:

```bash
backend
```

Execute:

```bash
./mvnw spring-boot:run
```

ou execute diretamente a classe:

```bash
ViaCartaoApplication.java
```

O backend iniciará na porta:

```bash
http://localhost:8080
```

---

# 🖥️ Frontend (ReactJS)

## Tecnologias utilizadas

- ReactJS
- Vite
- React Router DOM
- JavaScript
- Axios
- CSS Inline

---

## ▶️ Como executar o frontend

Abra a pasta:

```bash
frontend
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O frontend iniciará na porta:

```bash
http://localhost:5173
```
